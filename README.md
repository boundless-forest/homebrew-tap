# Boundless Forest Homebrew Tap

This tap distributes the Little Swan macOS menu bar app.

## Install

```sh
brew install --cask boundless-forest/tap/little-swan
```

Little Swan is ad-hoc signed and is not notarized by Apple. If macOS blocks the first launch, open **System Settings > Privacy & Security**, click **Open Anyway**, and confirm that you want to open Little Swan.

## Upgrade

```sh
brew upgrade --cask little-swan
```

## Uninstall

```sh
brew uninstall --cask little-swan
```

Source code and releases are available in the [Little Swan repository](https://github.com/boundless-forest/little-swan).
